# Changelog

All notable changes to WA Marketing Web Extension will be documented in this file.

## [1.0.0] - 2026-03-11

### Added
- Initial release of WA Marketing Web Extension
- Bulk message sending functionality
- Contact import from CSV/Excel
- Message personalization with dynamic fields ({{Name}}, {{Phone}}, etc.)
- Batch sending with configurable delay
- Export contacts from WhatsApp groups
- Export contacts from labels
- Export all contacts from chat list
- Real-time sending progress tracking
- Statistics dashboard (total, success, failed)
- Export sending results to CSV
- Retry failed messages
- Attachment support (images, videos, documents)
- Multiple file attachments
- Transparent sending process with progress bar
- Rate limiting to prevent WhatsApp blocks
- Random delay variance for human-like behavior

### Features
- Simple and intuitive UI
- Three main tabs: Simple, Export, Statistics
- Personalization field buttons for easy insertion
- File upload support for contact lists
- Configurable sending parameters
- Real-time status updates
- Failed message tracking and retry
- CSV export functionality

### Security
- No data sent to external servers
- All processing done locally in browser
- No storage of sensitive information
- Respects WhatsApp Web permissions

### Documentation
- Comprehensive README with usage instructions
- Installation guide for Chrome, Edge, and Firefox
- Troubleshooting section
- Best practices for avoiding WhatsApp blocks
- Example CSV format
